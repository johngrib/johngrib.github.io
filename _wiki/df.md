## 사용

```
$ df
```

### 옵션

* `-h` : `--human-readable`
* `-i` : `--inodes`. 아이노드 사용 정보를 확인.
* `-k` : `--kilobytes`. 1024 byte (1kb) 단위로 출력.
* `-a` : `--all`. 모든 파일 시스템을 출력.
