## Install
```sh
brew install ykman
```

## Examples
```sh
 # 도움말을 본다
ykman --help

 # 연결된 yubikey 목록을 본다
ykman list

 # 연결된 yubikey 중, 시리얼 번호가 0123456 인 yubikey의 정보를 본다
ykman --device 0123456 info
```
